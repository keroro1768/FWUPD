## 提案格式

建立新資料夾：`E001_xxx/`

```
E001_xxx/
├── Task.md       ← 強化構想
├── PLAN.md      ← 執行計畫
├── WORKLOG.md   ← 工作日誌
└── VERIFY.md   ← 完成後填寫
```

## 提案流程

1. 想到 Enhancement → 建立 `E0XX/` 資料夾
2. 填寫 `Task.md`
3. Dororo 審核價值
4. 確認後安排負責人
5. 執行，完成後填 `VERIFY.md`
